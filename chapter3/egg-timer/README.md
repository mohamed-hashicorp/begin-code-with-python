# Hello World in Python

## Introduction
This repository demonstrates a simple Python program that prints **"Egg timer"** to the console.
It is part of practicing beginner-level Python concepts inspired by the **Begin Code with Python** book and example materials here:
https://github.com/Begintocodewithpython/samples

## Prerequisites
Before running the code, ensure you have:
- **Python 3.x** installed (download: https://www.python.org/downloads/)
- A terminal or command prompt

Verify your Python installation:
```bash
python --version
# or
python3 --version
```

## Clone the code
Clone your repository and change into its directory:
```
git clone https://github.com/mohamed-hashicorp/begin-code-with-python
cd begin-code-with-python/chapter3/egg-timer
```

## Run the code
Run the script with:
```
python3 egg-timer.py
```
